# CI/CD Pipeline with Docker & GitHub Actions

## 📌 Project Overview
This project demonstrates a complete CI/CD pipeline using GitHub Actions to build and push a Dockerized Node.js application.

## 🚀 Tech Stack
- Node.js
- Docker
- GitHub Actions

## ⚙️ Features
- Containerized application using Docker
- Automated build and push using CI/CD pipeline
- DockerHub integration

## 🔄 CI/CD Workflow
1. Code pushed to GitHub
2. GitHub Actions triggered
3. Docker image built
4. Image pushed to DockerHub
